# App Frame

A handy way to have a header, footer, and flexible body that fills in-between.

## Usage

### Default

```html
<div class="tbds-app-frame">
  <div class="tbds-app-frame__header">
    Header
  </div>

  <div class="tbds-app-frame__body">
    Flexible body
  </div>

  <div class="tbds-app-frame__footer">
    Footer
  </div>
</div>
```
